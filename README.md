## About

Finding a workplace that respects focused, uninterrupted work is harder than it should be. This repository maintains a list of companies that actively embrace the principles of Deep Work -- making it a useful reference for developers, engineers, and knowledge workers who value concentration over constant connectivity.

This project was [discussed on Hacker News](https://news.ycombinator.com/item?id=25543989), where the community contributed additional suggestions.

Contributions are welcome -- open a pull request to add a company.

## What is Deep Work?

Cal Newport introduced the concept in his 2016 [book](https://www.amazon.com/Deep-Work-Cal-Newport-audiobook/dp/B0189PVAWY/ref=sr_1_1?dchild=1&keywords=deep+work&qid=1600695114&sr=8-1), though he had been writing about it on his blog since 2012:

> *"Professional activity performed in a state of distraction-free concentration that pushes your cognitive capabilities to their limit. These efforts create new value, improve your skill, and are hard to replicate."*

The opposite is **Shallow Work**:

> *"Non-cognitively demanding, logistical-style tasks, often performed while distracted. These efforts tend to not create new value in the world and are easy to replicate."*

For a shorter introduction, see [Doist's Complete Guide to Deep Work](https://doist.com/blog/complete-guide-to-deep-work/).
